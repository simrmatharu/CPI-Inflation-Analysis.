# CPI Inflation Analysis – India
This project analyzes India’s Consumer Price Index (CPI) data (2013–2023) to study inflation trends across sectors such as food, fuel, housing, and education.

## Tools Used
- Microsoft Excel
- Pivot Tables and Charts
- Dashboard Visualization

## Key Insights
- Identified major inflation-driving sectors.
- Compared rural and urban inflation patterns.
- Correlated CPI with oil price fluctuations.

## File
- `CPi project.xlsx` — contains data analysis and dashboard.
